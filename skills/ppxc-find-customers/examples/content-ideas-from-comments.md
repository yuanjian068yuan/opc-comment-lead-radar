# 示例：从评论线索生成内容选题

## 用户输入

```text
这批客户先不用跟进，你帮我看看他们最关心什么，写 3 条短视频选题。
```

## 输出示例

```text
这批评论里重复出现 3 个内容角度：

1. 价格犹豫
钩子：不是你不想学英语，是你怕花钱后坚持不下来。
口播：很多人问“报了课会不会浪费”。先别急着买长期课，先做一次口语测评，再用 7 天看自己能不能坚持。
行动引导：评论区打“测评”，我帮你判断适合哪种练法。

2. 效果怀疑
钩子：上班以后练口语，到底还有没有用？
口播：如果你要面试、汇报、出差，口语不是考试，是工作工具。先练 20 个高频工作场景，比背单词更快见效。
行动引导：告诉我你的工作场景，我给你 3 句开口模板。
```

## 注意

内容选题要引用客户原话里的真实表达，不要写成泛泛营销口号。
