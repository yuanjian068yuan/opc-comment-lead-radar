# 示例：B2B 税务咨询找客户

## 用户输入

```text
我做企业税务咨询，客户是小微企业老板。帮我从快手评论里找真实咨询需求。
```

## 输出示例

```text
这轮我没有继续硬搜“税务咨询”这个泛词，而是换成 3 个问题词。共看了 6 条内容、118 条公开评论，筛出 5 个可能有咨询需求的老板。

1. 做建材的小周（高意向 · 风险咨询）：“公转私以前没注意，现在被提醒了，这种要怎么补救？”
   跟进话术：先问公司类型和发生时间，再建议做一次风险梳理。
2. 阿强门店（中意向 · 政策不懂）：“小规模这个季度突然补这么多，是不是哪里算错了？”
   跟进话术：先让他确认开票额和申报周期，不要直接报价。
```

## 注意

B2B 服务要找“具体问题 + 可能付费咨询”的评论，不要把同行讨论当客户。
